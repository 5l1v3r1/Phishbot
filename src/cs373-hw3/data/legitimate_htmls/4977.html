12th May 2016

PostgreSQL 9.6 Beta 1, 9.5.3, 9.4.8, 9.3.13, 9.2.17 and 9.1.22 Released!

The PostgreSQL Global Development Group is pleased to announce the availability of PostgreSQL 9.6 Beta 1, the first beta release of the upcoming 9.6 version of PostgreSQL.

Please download and test the beta code with your applications, and report any issues you may find.


9.6 Beta 1 Release Announcement
9.6 Beta 1 Release Notes
Download

We are also pleased to announce the availability of PostgreSQL 9.5.3, 9.4.8, 9.3.13, 9.2.17 and 9.1.22.

These are update releases fixing a number of issues that have been reported. Users are encouraged to upgrade as soon as practical.
