Stratus MLS requires JavaScript to be enabled.

To Enable:

  1. Click on the 'Tools' menu on your browser's toolbar.
  2. Select 'Internet Options'.
  3. Click on the security tab.
  4. Set your security level to medium.
  5. Click on the 'OK' button.
 
User ID = Membership #
By clicking on the "Log On" button you are confirming that the Toronto Real Estate Board has authorized your use of the User ID, PIN and Password entered and that you are an Authorized User. Fraudulently accessing the system using someone else's User ID, PIN, and Password are criminal offences punishable by up to ten years imprisonment and may result in statutory and contractual damages.
