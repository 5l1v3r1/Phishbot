Purchase an access code

Great idea! Let’s get you those keys to success (aka LearnSmart and SmartBook).

Go to www.MHEducation.com

Search for the product you need by using the search bar at the top of the page.

A) First select Higher Ed under the select drop down menu.

B) Then, enter your LearnSmart or SmartBook ISBN into the search bar. That’s the 10-digit number associated with the product. No worries if you don’t know the ISBN — you can search for the product using its title and edition number instead.

Once you’ve found the product that matches your search criteria, add the product to your cart.
Before checking out, we will ask you to sign in. For those of you who are new to our products, you must create an account. Make sure to use a valid email address and a password you can easily remember.

If you run into any technical issues along the way, please give us a call at (800) 331-5094.

Register an access code

Congrats! You are the proud owner of a truly unique 20 digit alphanumeric code, which probably looks something like this: 48ZO-00BJ-T9IJ-JZW5-2R2A… Now what?

First, go to www.MHEducation.com

You can easily find the product you’ve purchased by using the search bar at the top of the page.

A) Select Higher Ed under the select drop down menu.

B) Enter the access code you purchased for your course into the search bar.

Click Redeem to access the product.

First time using our products? No big. Create an account and get ready for greatness.
Old pro? Welcome back! Make sure to sign in using the email and password you’ve used for previous courses.

If you run into any technical issues along the way, please give us a call at (800)331-5094.

Access my products

First step: give yourself a high-five. You’re ahead of the game.

After signing into www.MHEducation.com click My Account. You’ll find this located in the top right hand corner.

Scroll down the page to My Digital Products. Press the Launch button next to the LearnSmart or SmartBook product you have purchased and you’re good to go.

If you run into any technical issues along the way, please give us a call at (800)331-5094.
