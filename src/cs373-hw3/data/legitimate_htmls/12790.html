3D Secure (ou 3DS) ?
3D Secure (également appelé "Verified by Visa" chez Visa et "MasterCard SecureCode" chez Mastercard) est un système de paiement par authentification qui permet de sécuriser vos paiements lors d'achats en ligne.
Lors du paiement, votre banque vérifie l'identité du porteur de la carte avant de valider la transaction pour éviter les paiements frauduleux par cartes bancaires (Exemple : numéros de cartes volés).
Fonctionnement
Après avoir effectué vos achats sur internet comme d'habitude et renseigné vos coordonnées bancaires, vous êtes transféré sur le site de l'organisme bancaire en charge de la transaction.
Des informations complémentaires, différentes selon les banques, vous sont alors demandées.
Il peut s'agir de :
  • Saisir votre date de naissance
  • Renseigner un code reçu par sms
  • Répondre à une question personnelle
Cette étape supplémentaire permet de garantir l'identité de l'utilisateur de la carte via le renseignement d'informations que le détenteur est le seul à connaitre.

Pour toute question sur le code 3D Secure (obtention, perte, modification...), contactez directement votre banque.

ATTENTION : après 3 échecs d'authentification, votre transaction sera annulée.
Votre carte sera bloquée au bout de 3 transactions bancaires annulées. Contactez alors votre banque.
