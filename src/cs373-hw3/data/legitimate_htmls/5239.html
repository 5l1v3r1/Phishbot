HighPoint RocketStor 6324L
20607 руб.
купить
Описание
Внешний SATA контроллер систем хранения с двумя портами Thunderbolt 2 для подключения к Mac Pro, MacBook Pro, iMac и компьютерам на Windows с портами Thunderbolt внешних четырехдисковых хранилищ с интерфейсом mini-SAS (SFF-8088), а также однодисковых eSATA накопителей (необходим кабель HighPoint EXT-MS-1MEJ). По сути, RocketStor 6324L является переходником с eSATA на Thunderbolt 2 или c mini-SAS на Thunderbolt 2, но только для коробок с SATA дисками.
