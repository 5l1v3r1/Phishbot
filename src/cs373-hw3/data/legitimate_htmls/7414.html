  2. Skylake Status Update 10.11.4

    tonymacx86 | 05-06-16 03:15:23 PM

    Great news! As of 10.11.4, Skylake systems do not need special configuration (Piker Alpha's Skylake AppleAPIC Patch) to boot. Furthermore, Intel HD 530 integrated graphics are now working without artifacts. Default UniBeast created USB drives created with 10.11.4 can boot the installer without any changes. Also, MultiBeast can now be used for full post-installation setup.
