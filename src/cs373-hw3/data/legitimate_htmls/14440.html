Jssor Slider is touch swipe image slider carousel with 390+ caption slide effects. When touch Jssor Slider, it will freeze and then move to the direction that finger swipes to. It comes with javascript library source code.

Jssor Slider is reponsive by calling API. The API method '$ScaleWidth' is the best performance responsive slider carousel approach, Jssor Slider can be scaled to any size at realtime.
